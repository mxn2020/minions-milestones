---
title: Installation
description: Install Minions Milestones packages
---

## TypeScript / Node.js

```bash
npm install @minions-milestones/sdk minions-sdk
```

## Python

```bash
pip install minions-milestones
```

## CLI

```bash
npm install -g @minions-milestones/cli
```

## Requirements

- **Node.js** 20 or later
- **Python** 3.11 or later (for Python SDK)
