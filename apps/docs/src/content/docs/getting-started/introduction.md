# Minions Milestones

Milestone definitions, dependencies, and completion criteria

## Overview

This project is part of the [Minions ecosystem](https://github.com/mxn2020/minions) and provides:

- **TypeScript SDK** (`@minions-milestones/sdk`) — Core library for Node.js
- **Python SDK** (`minions-milestones`) — Python bindings
- **CLI Tool** (`@minions-milestones/cli`) — Command-line interface
- **Web App** — Interactive playground at [milestones.minions.wtf](https://milestones.minions.wtf)
